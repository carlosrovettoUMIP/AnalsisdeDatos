# 🗺️ Recursos Externos y Fuentes de Datos

Para profundizar en el análisis de datos marítimos, se recomiendan las siguientes fuentes oficiales y herramientas adicionales.

## 🚢 Autoridades Marítimas
- **Autoridad Marítima de Panamá (AMP)**: [amp.gob.pa](https://amp.gob.pa)
- **Autoridad del Canal de Panamá (ACP)**: [pancanal.com](https://pancanal.com)
- **IMO GISIS**: [gisis.imo.org](https://gisis.imo.org) - Registros globales de buques.

## 📊 Fuentes de Datos Abiertos (Datasets)
- **MarineTraffic Open Data**: Muestras históricas de AIS.
- **Global Fishing Watch**: Datos de actividad pesquera global.
- **UNCTAD Statistics**: Estadísticas de comercio y transporte marítimo.
- **World Bank Port Data**: Indicadores de rendimiento portuario (LPI).

## 🛠️ Herramientas Recomendadas
- **Python libraries**: Geopandas (Mapas), Plotly (Interactividad), PySpark (Big Data).
- **QGIS**: Para análisis espacial avanzado de rutas navieras.
- **Google Colab**: Entorno de ejecución en la nube gratuito.
